# Metadata Standards

## Candidate ID

Format:

CAND_0001
CAND_0002

---

## Job ID

Format:

JOB_0001
JOB_0002

---

## Model Version

Format:

v1.0
v1.1
v2.0

---

## Timestamp

Format:

YYYY-MM-DD HH:MM:SS

Example:

2026-05-30 19:30:00

---

## Metadata Example

{
  "candidate_id": "CAND_0001",
  "job_id": "JOB_0001",
  "model_version": "v1.0",
  "timestamp": "2026-05-30 19:30:00"
}
