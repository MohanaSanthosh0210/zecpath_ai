# Confidence Analyzer

## Overview

The Confidence Analyzer estimates a candidate's confidence level during an AI interview using behavioral indicators extracted from the candidate's response.

Instead of relying on subjective human judgement, the analyzer evaluates measurable hesitation patterns and repeated speech to produce a normalized confidence score.

The module automatically consumes the structured answer generated by the Answer Understanding module (Day 25).

---

# Input

The analyzer reads the candidate response from:

data/understood_answers/sample_answer.json

Specifically:

structured_answer.text

---

# Behavioral Indicators

The analyzer evaluates:

- Hesitation phrases
- Uncertainty expressions
- Repeated consecutive words

---

# Hesitation Phrases

Examples include:

- I think
- Maybe
- Perhaps
- Probably
- Not sure
- Kind of
- Sort of
- I guess
- I believe

Each detected hesitation lowers the confidence score.

---

# Repeated Words

The analyzer detects repeated consecutive words such as:

"I worked worked on Django."

Repeated words may indicate hesitation or uncertainty.

---

# Confidence Score

The analyzer produces a normalized score between 0 and 100.

Higher scores indicate stronger confidence.

---

# Generated Output

Example:

{
    "confidence_score": 90,
    "hesitation_count": 1,
    "repeated_words": 0
}

This output becomes one component of the overall Behavioral Confidence Score.
