# Integrity Detection Framework

Version: 1.0

---

# Objective

Design a non-invasive AI framework capable of identifying possible malpractice or external assistance during interviews.

The framework provides explainable integrity signals without making automatic accusations.

---

# Behavioral Signals

The system observes:

- Tab Switching
- Screen Focus Loss
- External Voice
- Repeated Gaze Deviation
- Long Response Pause

---

# Detection Strategy

The framework combines:

- Threshold-based detection
- Pattern recognition
- Behavioral context

to estimate interview integrity.
