# Technical Scoring Engine Architecture

Version: 1.0

---

# Objective

The Technical Scoring Engine evaluates a candidate's technical interview responses using explainable, weighted, and difficulty-aware scoring.

Unlike keyword-based evaluation, this module scores multiple dimensions independently and combines them into a unified technical score.

---

# Inputs

The engine consumes structured technical answer data generated by later Technical Interview modules.

Expected Input

- Candidate ID
- Question ID
- Question Difficulty
- Accuracy Score
- Depth Score
- Logical Reasoning Score
- Real-world Applicability Score

---

## Applicability Scorer

## Difficulty Normalizer

Easy

↓

## Technical Scoring Engine
