# Fairness Review Notes

## Objective

Evaluate whether the HR Interview AI introduces unfair bias into candidate evaluation.

---

# Audit Scope

Modules reviewed:

- ATS Evaluation
- Resume Screening
- HR Interview
- Communication Evaluation
- Confidence Evaluation
- Aptitude Evaluation
- Unified Scoring Engine

---

# Protected Attributes

The following demographic attributes are excluded from scoring:

- Age
- Gender
- Religion
- Race
- Ethnicity
- Nationality
- Marital Status
- Disability
- Political Affiliation

---

# Scoring Features

Candidate evaluation is based on:

- Technical Skills
- Experience
- Resume Quality
- Communication
- Logical Reasoning
- HR Responses
- Confidence
- Problem Solving

---

# Fairness Findings

✓ No prohibited demographic fields are used during scoring.

✓ Candidate evaluation is competency-based.

✓ Scores are generated using predefined weighting rules.

✓ Unified Scoring Engine combines objective evaluation metrics.

---

# Residual Risks

Potential risks include:

- Speech recognition errors
- Accent variability
- Very short responses
- Extremely noisy audio

These risks are partially mitigated using transcript cleanup and confidence normalization.

---

# Recommendation

Continue periodic fairness audits after deployment.
