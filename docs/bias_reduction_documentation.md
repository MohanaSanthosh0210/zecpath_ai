# Zecpath AI Bias Reduction Strategy

## Objective

Reduce hiring bias and ensure fair candidate evaluation.

## Methods

### Resume Normalization

All resumes are converted into a common structure before evaluation.

### Personal Attribute Masking

The following attributes are ignored:

- Name
- Gender
- Age
- Religion
- Nationality
- Marital Status

### Semantic Matching

Candidate evaluation is based on skills, experience, education, and project relevance rather than exact keyword frequency.

### Score Normalization

Scores are normalized to a common 0–100 scale to ensure consistency across job roles.

## Outcome

Improved fairness, consistency, and explainability of ATS scoring.
