# Technical Scoring Rubric

Version: 1.0

---

# Objective

The Technical Scoring Engine evaluates a candidate's technical responses using explainable and configurable scoring criteria.

The engine focuses on both correctness and engineering thinking rather than keyword matching alone.

---

# Scoring Parameters

## 1. Accuracy (35%)

Measures whether the candidate answered the technical question correctly.

Evaluation Criteria

- Correct concepts
- Correct terminology
- Correct implementation approach
- Minimal factual errors

Score Range

0–100

---

## 2. Depth of Explanation (30%)

Measures how thoroughly the candidate explains the solution.

Evaluation Criteria

- Detailed explanation
- Understanding of underlying concepts
- Ability to explain technical decisions
- Completeness

Score Range

0–100

---

## 3. Logical Reasoning (20%)

Measures structured thinking.

Evaluation Criteria

- Step-by-step reasoning
- Cause-and-effect explanation
- Clear technical flow
- Decision justification

Score Range

0–100

---

## 4. Real-world Applicability (15%)

Measures engineering practicality.

Evaluation Criteria

- Production considerations
- Performance
- Scalability
- Security
- Maintainability

Score Range

0–100

---

# Overall Technical Score

The final score is calculated using weighted scoring.

Final Score

=

Accuracy × Weight

+

Depth × Weight

+

Reasoning × Weight

+

Applicability × Weight

---

# Difficulty Normalization

Scores are normalized based on question difficulty.

Easy

↓

Medium

↓

Hard

↓

Expert

This prevents unfair comparison between candidates answering different difficulty levels.

---

# Explainable Evaluation

The scoring engine provides:

- Individual parameter scores
- Overall technical score
- Strengths
- Weaknesses
- Improvement suggestions

---

# Output

The Technical Scoring Engine generates:

- Skill-wise score breakdown
- Technical evaluation report
- Explainable scoring summary
