# AI Screening System Documentation

## Overview

The AI Screening System is an end-to-end AI-powered candidate screening platform developed to automate the recruitment process. It integrates resume analysis, ATS scoring, interview response understanding, behavioral analysis, conversation management, testing, optimization, and edge-case handling into a single modular pipeline.

The objective of the system is to reduce manual recruiter effort while providing consistent, explainable, and data-driven candidate evaluations.
