# Recruiter Report Format

The AI Screening Report contains:

- Candidate ID
- Overall Screening Score
- Overall Status
- Communication Strength
- Confidence Score
- Sentiment
- Key Answers
- Skills Confirmed
- Experience
- Salary Expectation
- Availability
- Strengths
- Risks
- Missing Information

Output Format: JSON

Example:

{
    "candidate_id": "...",
    "overall_status": "...",
    ...
}
