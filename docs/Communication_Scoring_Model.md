# Communication Scoring Model

## Overview

The Communication Scoring Model evaluates a candidate's communication skills during the AI interview process.

Unlike a standalone communication evaluator, this module is fully integrated with the AI screening pipeline. It automatically reads the structured candidate response generated by the Answer Understanding module (Day 25) and computes objective communication metrics.

The model produces a normalized communication score between 0 and 100 while minimizing subjective bias.

# Evaluation Components

## Fluency

Measures:

- Sentence continuity
- Speaking smoothness
- Average sentence length
- Natural explanation flow

Output:

- Fluency Score

---

## Grammar Quality

Measures:

- Sentence capitalization
- Punctuation
- Basic grammar correctness
- Writing consistency

Output:

- Grammar Score

---

## Vocabulary Range

Measures:

- Total words
- Unique words
- Vocabulary diversity
- Word richness

Output:

- Vocabulary Score

---

## Clarity

Measures:

- Number of complete sentences
- Explanation completeness
- Logical flow
- Average sentence length
- Supporting information

Output:

- Clarity Score

---

## Filler Word Detection

Detects unnecessary filler words such as:

- um
- uh
- like
- actually
- basically
- literally
- you know
- kind of
- sort of

Output:

- Filler Score

---

## Answer Structure

Measures whether the response contains:

- Introduction
- Supporting explanation
- Conclusion

The evaluator does not depend only on keywords but also analyzes sentence organization.

Output:

- Structure Score

---
